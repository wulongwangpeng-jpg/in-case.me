## 一、海外站（wanyine.com）

### 部署架构
```
Vercel（美国） + Supabase 新加坡 PostgreSQL
→ OpenAI + Twilio 短信 + Resend 邮件 + Stripe 支付
→ 纯订阅，无免费功能
```

### 成本与临界点

| 阶段 | 触发条件 | 月成本 | 年成本 |
|------|---------|--------|--------|
| 上线前 | — | $0 | $12（域名） |
| 0-100 付费用户 | 全在 Free Tier | $0 | $12 |
| 100+ 付费用户 | Supabase 500MB 满 | $25 | $312 |
| 500+ 付费用户 | Vercel 带宽超 100GB | $45 + 短信按量 | ≈ $552+ |

> 全部按付费用户计，因为无免费用户。Supabase 是第一个触发付费的服务。

### 收入模型（$99/年）

| 付费用户 | 年收入 | 年成本 | 利润 |
|---------|--------|--------|------|
| 10 | $990 | $12 | ✅ |
| 50 | $4,950 | $12 | ✅ |
| 100 | $9,900 | $312 | ✅ |
| 500 | $49,500 | $552 | ✅ |

### 合规清单

| 事项 | 状态 | 说明 |
|------|------|------|
| GDPR（欧盟用户） | 🔴 要做 | 隐私政策 + Cookie 弹窗 + 数据删除权 |
| CCPA（加州用户） | 🔴 要做 | 隐私政策加「不卖数据」声明 |
| Stripe 商户验证 | 🔴 要做 | 身份验证 + 绑银行卡收款 |
| 数字遗产法律免责 | 🔴 要做 | 服务条款写清楚「非法律文书」 |
| 服务条款 + 隐私政策 | 🔴 要做 | 上线前必须挂 |
| 美国公司注册 | 🟡 建议 | Stripe 可能支持中国个人户，长期建议 LLC |
| ICP 备案 | ❌ 不需要 | 服务器在海外 |

## 二、国内站（wanyine.cn）

### 部署架构
```
阿里云 ECS + 阿里云 RDS PostgreSQL
→ DeepSeek + 阿里云短信 + 阿里云邮件推送
→ 免费，无支付系统
```

### 月费

| 项目 | 月费 |
|------|------|
| 阿里云 ECS（2vCPU 4G） | ¥100-200 |
| 阿里云 RDS PostgreSQL 基础版 | ¥200-300 |
| 阿里云短信 | 按量 ¥0.045/条 |
| 阿里云邮件推送 | 按量 ¥0.0015/封 |
| 域名 wanyine.cn | ¥29-69/年 |
| **合计** | **≈ ¥350-550/月** |

### 合规清单

| 事项 | 状态 | 说明 |
|------|------|------|
| ICP 备案 | 🔴 必须 | 阿里云代交，个人可备案 |
| 公安备案 | 🔴 必须 | ICP 通过后 30 天内 |
| 个人信息保护法 | 🔴 要做 | 隐私政策 + 用户同意 + 删除权 |
| 短信签名备案 | 🔴 要做 | 文案不能含死亡/遗产/失联——中性化文案已规避 |
| 增值电信许可证 | ✅ 不触发 | 纯免费，不构成经营性质 |
| 服务条款 + 隐私政策 | 🔴 要做 | 上线前必须挂 |

> **核心判断**：国内站不收钱 → 个人 ICP 备案够用。一旦收钱 → 需要企业备案 + ICP 经营许可证，个人开发者拿不到。

## 三、两站关系

| | 海外 | 国内 |
|--|------|------|
| 模式 | 纯订阅付费 | 完全免费 |
| 收入 | ✅ 收入来源 | ✗ 无收入 |
| 成本风险 | 几乎零成本启动 | 月 ¥400-500 固定支出 |
| 合规难度 | 低 | 中 |

> 5 个海外付费用户 = $495/年 ≈ ¥3,500，够覆盖国内站一年服务器成本。国内站的 ¥500/月是营销费用——用国内免费体验培养海外潜在用户。

## 四、上线节奏

```
阶段一：海外站上线（6-8 天）
├── 中英字符串分离 + 英文文案 + 品牌叙事重设计
├── 通知层接入 Twilio + Resend
├── 服务商抽象层（AI/短信/邮件按域名切）
├── Stripe 订阅系统
├── 执行清单剩余项
└── Product Hunt 发布

阶段二：验证（1-3 个月）
├── 看 PH 数据 + 付费转化率
├── 收集英文用户反馈
└── 迭代英文文案调性

阶段三：国内站上线（3 天 + 两周备案等待）
├── 阿里云 ECS + RDS 搭建
├── ICP 备案
├── 国内服务商接入
└── 国内免费上线
```

## 五、容易被忽略的点

1. **英文文案不是翻译**。中文「温暖地交代一切」不能直译——海外品牌叙事需要独立设计，是阶段一最大工作量。

2. **Stripe 中国身份收款**。需提前确认中国身份证 + 中国银行卡能否开通 Stripe 商户。不支持则需替代方案（Lemon Squeezy / Paddle，费率更高）。

3. **客服成本**。海外用户有时差，英语邮件客服。0 用户不用管，用户多后是时间黑洞。建议上线时写英文 FAQ 页面减少重复回答。

4. **Product Hunt 不是发了就完**。需要提前准备：Hunter 合作、预热注册、首小时投票策略、英文截图和视频。24 小时窗口错过就没了。

5. **数据备份**。Supabase 免费版自动备份需确认，正式上线前做好保障。
